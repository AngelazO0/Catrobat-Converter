Convertidor de versiones .catrobat

Esta herramienta te permite modificar la versión de un archivo .catrobat para hacerlo compatible con versiones anteriores o más recientes.




¿Cómo funciona?

El archivo .catrobat contiene un archivo interno llamado code.xml. Esta herramienta busca y modifica la versión dentro de ese archivo.
- Si seleccionas una versión diferente en ambas listas, la herramienta cambiará los valores correspondientes.
- Si intentas convertir el archivo a la misma versión, recibirás un aviso.
